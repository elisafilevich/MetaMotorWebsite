Algorithms to quantify Metacognition

The current gold standard to measure metacognitive ability is Meta-d'. This method requires discrete confidence ratings. Because we often use quasi-continuous confidence ratings, we wonder if there is a way to optimize, or perhaps even eliminate, the need to discretize continuous confidence ratings into finite bins.

We work on this project together with Guillermo Solovey.
